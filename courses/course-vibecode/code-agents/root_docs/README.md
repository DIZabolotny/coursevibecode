## Как читать

1. Для быстрого старта читай сначала короткие вводные `*_readme.md`, затем learning paths.
2. Для настройки среды переходи к документам про settings, permissions и memory hierarchy.
3. Для выбора инструмента используй comparative-документы.
4. Для более глубокой системной картины переходи к полным management-гайдам.
