**Codex**
Если нужен один главный вход, то это официальный хаб OpenAI:
- [Codex docs hub](https://developers.openai.com/codex)

Это сейчас самый близкий вариант к “полноценному руководству”: в одном месте собраны разделы по `App`, `IDE Extension`, `CLI`, `Web`, `Configuration`, `Automation`, `Security`, `Best practices`, `Cookbooks`, `Changelog` и `Open Source`.[^codexhub]

Самые полезные страницы внутри:

- [Best practices](https://developers.openai.com/codex/learn/best-practices)

- [Command line options](https://developers.openai.com/codex/cli/reference)

- [Codex Prompting Guide](https://developers.openai.com/cookbook/examples/gpt-5/codex_prompting_guide)
- [Open Source / CLI repo pointer](https://developers.openai.com/codex/open-source)

- [openai/codex on GitHub](https://github.com/openai/codex)

**Claude Code**
Главный официальный хаб:
- [Claude Code docs home](https://code.claude.com/docs/en/)

Это у Anthropic тоже основной “единый вход”: там есть `Getting started`, `Build with Claude Code`, `Deployment`, `Administration`, `Configuration`, `Reference`, `Resources`, плюс отдельные разделы по `Best practices`, `Common workflows`, `VS Code`, `JetBrains`, `Web`, `Slack`, `Remote Control` и т.д.[^claudehub]

Ключевые страницы:
- [Overview](https://code.claude.com/docs/en/overview)
- [Advanced setup](https://code.claude.com/docs/en/getting-started)
- [Settings](https://code.claude.com/docs/en/settings)
- [CLI reference](https://code.claude.com/docs/en/cli-reference)
- [Common workflows](https://code.claude.com/docs/en/common-workflows)
- [Best practices](https://code.claude.com/docs/en/best-practices)
